## 2. Scope

**Mechanical cleanup** — sweep affected tests, rewrite к race-free
patterns. Verify regression baseline restored.

**Race-free patterns** (per Plan 83.6 commit lesson):
1. **`parallel for i in 0..N { body }`** — returns `[]T`, each fiber
   writes к own slot. No shared mut.
2. **Channel-based aggregation** — fibers send results, single consumer
   receives + aggregates. No shared mut.
3. **Atomic counters** (если doesn't break test semantic) — verify
   monotonicity, not exact ordering.

**НЕ В SCOPE:**
- Runtime modifications (никаких C/Rust changes).
- New runtime features (atomics, channels).
- Performance benchmarks.
- Test directive changes (`// ENV NOVA_AUTOARM=0` — НЕ удалять без verification).

## 3. Affected tests detail

### 3.1 `concurrency/supervised_errors` — "early-stop pattern via shared error flag"

**Current (race-prone):**
```nv
test "supervised: early-stop pattern via shared error flag" {
    let mut aborted = false
    let mut work_done = 0
    supervised {
        spawn {
            if !aborted { … aborted = true … else work_done += 1 … }
        }
        // ... more spawns читают/пишут aborted + work_done
    }
    assert(work_done == 0)
}
```

**Race:** N fibers concurrently read `aborted` + write `aborted/work_done`.
Pre-83.7 deque LIFO scheduling = effectively sequential; runnext + 83.10
fix changed ordering → race surfaces, `work_done > 0` либо random.

**Rewrite strategy:** use `parallel for` returning `[]bool` либо
`[]int` per-slot status. Sequential aggregation after.

Open file, identify which sub-test. Pattern likely shared-mut counter

### 3.3 `concurrency/mn_runtime_smoke` — 1/4 sub-tests fail

### 4.5 Rewrite examples

**Pattern 1: parallel for replacement**

Before:
```nv
let mut counter = 0
supervised {
    spawn { counter += 1 }
    spawn { counter += 1 }
}
assert(counter == 2)
```

After:
```nv
let xs = parallel for i in 0..2 { 1 }
let mut counter = 0
let mut i = 0
while i < 2 {
    counter = counter + xs[i]
    i = i + 1
}
assert(counter == 2)
```

**Pattern 2: channel aggregation** (когда parallel-for не подходит,
e.g. dynamic spawn count):

Before:
```nv
let mut aborted = false
let mut work_done = 0
supervised {
    spawn { if some_cond { aborted = true } else { work_done += 1 } }
}
```

After:
```nv
let ch = Channel.new(N)
supervised {
    spawn {
        if some_cond { ch.tx.send(0) } else { ch.tx.send(1) }
    }
}
// aggregate after supervised exit
let mut work_done = 0
let mut got = 0
while got < N {
    let v = match ch.rx.recv() { Some(x) => x, None => 0 }
    work_done = work_done + v
    got = got + 1
}
```

**Pattern 3: keep cooperative directive** (если test inherently
cooperative-dependent, e.g. exact ordering required):

Если rewrite ломает test semantic — добавить `// ENV NOVA_AUTOARM=0`
directive + comment explaining почему.
