## Migration candidates

### 1. `nova_tests/plan90/copy_from.nv` — test "src короче dst (хвост не тронут)"

**Current** (tests old truncation-silent semantics):
```nv
test "copy_from — src короче dst (хвост не тронут)" {
    let mut dst []u8 = [1, 2, 3, 4]
    let src []u8 = [9, 9]
    dst.copy_from(src)
    let expect []u8 = [9, 9, 3, 4]
    assert(dst.compare(expect) == 0)
}
```

**Action:** Replace with slice-idiom version (explicit prefix):
```nv
test "copy_from — slice prefix (dst[..n].copy_from(src))" {
    let mut dst []u8 = [1, 2, 3, 4]
    let src []u8 = [9, 9]
    dst[..2].copy_from(src)
    let expect []u8 = [9, 9, 3, 4]
    assert(dst.compare(expect) == 0)
}
```
