## Цель
GC обходит **shadow-цепочку + unified registry** вместо консервативного скана C-стеков/
fiber-arena. **Swap вершины** `nova_shadow_top` на fiber-switch (сохранение в состояние fiber'а
при park, восстановление при resume — аналог NT_TIB.StackBase).

- Boehm stack-scan **отключён** (свой root-provider).
- Полная регрессия зелёная.
- **Закрывает [reference-mn-race]** (fiber-stack scan больше НЕ консервативен — снимает
  Windows `SuspendThread` conservative-скан minicoro-стеков, корень M:N-race'ов, Plan 83.11
  §12.23): стресс-фикстура M:N зелёная **без `GC_DONT_GC`**.
- «без упрощений как для прода» — обязательный критерий.
