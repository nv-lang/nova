### Ф.3 — LSP quick-fixes (Plan 50 D102 integration)

**Acceptance:** все 4 error codes из Plan 101.1 + warnings — имеют
machine-applicable suggestions в Plan 50 D102 format.

Format:
```json
{
  "code": "E_BARE_TYPEVAR_NEEDS_PREFIX",
  "message": "bare typevar `T` requires `fn[T]` prefix",
  "suggestions": [
    {
      "label": "Add `fn[T]` prefix",
      "edit": { "range": "fn T @method", "text": "fn[T] T @method" }
    },
    {
      "label": "Use as named type (define `type T`)",
      "edit": { "range": "...", "text": "type T { ... }\n\nfn T @method" }
    }
  ]
}
```

Files: `compiler-codegen/src/diagnostic/...`
