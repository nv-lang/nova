## Цель
Снять **8MB→малый reserve** на fiber. **Развилка (§7.6 + H5):**
- **segmented** (добавить чанк при нехватке) — НЕ нужен moving, работает с non-moving precise
  ИЛИ даже conservative → **основной путь** (учитывая moving-вердикт §7.6);
- **copying** (релокация стека) — нужен moving + **блокируется H5** (замыкания захватывают по
  указателю В кадры → копирование инвалидирует, если не чинить by-value-захватом из
  [144.0.1](144.0.1-gate-frame-abi-object-start.md)).

Целить **segmented**; copying — только если решён H5 и доказана нужда.

## Acceptance criteria
- **100k+ fiber'ов** в бюджете (снимает потолок fiber_arena Plan 82/149).
- **segmented-grow зелёный** без релокации указателей.
- «без упрощений как для прода» — обязательный критерий.
