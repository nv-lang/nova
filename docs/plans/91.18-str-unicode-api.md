### Nova синтаксис — критично

```
priv          = module-private (видны пирам, не видны снаружи)
priv(type)    = type-private (только методы того типа)
ro            = immutable binding
mut           = mutable binding
@             = self в методах
-> @          = fluent return (возвращает self для chaining)
export fn T @m() = extension-метод на тип T
```

**Запрещено:** `;` как разделитель операторов. Каждый оператор — на отдельной строке.

**Tuple binding:** `ro (a, b) = (expr1, expr2)` — НЕ две отдельные `ro a = ...; ro b = ...`.

**Объявление типа:** `export type Foo value priv(type) { field Type }` (value = stack,
priv(type) = поля доступны только методам Foo).

**Contracts:** `requires condition` — precondition, elided at compile-time если Z3 доказывает.

**#no_prelude** — pragma в файлах `std/runtime/string/` (нельзя импортировать prelude рекурсивно);
вместо этого явный import `std.prelude.core.{...}`.

### Б1. `@trim*` — неверный критерий WS

**Файл:** `std/runtime/string/transform.nv`

Найти все три функции `@trim()`, `@trim_start()`, `@trim_end()`. В каждой цикл

```nova
// СТАЛО:
while start < n && is_ascii_ws(bytes[start] as int) { start = start + 1 }
```

### Б3. `@rfind()` — прямой скан без early exit

**Файл:** `std/runtime/string/search.nv`

```nova
// БЫЛО (O(n) всегда, нет early exit):
export fn str @rfind(needle str) -> Option[int] {
    ro sn = @byte_len()
    ro nn = needle.byte_len()
    if nn == 0 { return Some(sn) }
    if nn > sn { return None }
    mut last = -1
    mut i = 0
    while i + nn <= sn {
        if unsafe { RawMem.compare(@ptr + i, needle.ptr, nn) == 0 } { last = i }
        i = i + 1
    }
    if last < 0 { None } else { Some(last) }
}

// СТАЛО (backward scan, O(1) best case):
export fn str @rfind(needle str) -> Option[int] {
    ro sn = @byte_len()
    ro nn = needle.byte_len()
    if nn == 0 { return Some(sn) }
    if nn > sn { return None }
    mut i = sn - nn
    while i >= 0 {
        if unsafe { RawMem.compare(@ptr + i, needle.ptr, nn) == 0 } { return Some(i) }
        i = i - 1
    }
    None
}
```

## Ф.3 — Ленивый `WordsIter`

Цель: `@as_words() -> WordsIter` становится O(1). Сейчас `WordsView.new` делает
три O(n) прохода и хранит три Vec. Заменяем forward state machine.

**Файл:** `std/unicode/words.nv`

### 3.1 Новый тип

Заменить:
```nova
export type WordsView value priv(type) {
    buf str
    bounds Vec[int]
    idx int
}
fn WordsView.new(buf str) -> WordsView => { buf, bounds: word_boundaries(buf), idx: 0 }
```

На:
```nova
export type WordsIter value priv(type) {
    buf str
    pos int               // байт-офсет начала следующего сегмента
    n int                 // buf.byte_len()
    prev_imm_cat int      // категория непосредственно предыдущего cp (для WB3/WB3c/WB3d/WB4)
    prev_eff_cat int      // последний non-ignorable (effective prev, для WB5+)
    prev_prev_eff_cat int // предшествующий prev_eff_cat (для WB7/WB7c/WB11)
    ri_run int            // счётчик Regional_Indicator (WB15/WB16)
}

fn WordsIter.new(buf str) -> WordsIter =>
    { buf, pos: 0, n: buf.byte_len(),
      prev_imm_cat: 0, prev_eff_cat: 0, prev_prev_eff_cat: 0, ri_run: 0 }
```

### 3.2 `wb_is_break_s` — flat-параметрическая версия

Добавить рядом с `wb_is_break` (которая остаётся для `to_titlecase`):

```nova
// Flat-parameter версия wb_is_break для ленивого итератора.
// Параметры вместо random-access в массивы:
//   p_imm  — immediate prev cat (WB3/WB3c/WB3d/WB4)
//   pcat   — effective prev cat (последний non-ignorable, WB5+)
//   ppcat  — до pcat (WB7/WB7c/WB11)
//   c      — cur cat
//   cur_cp — cur codepoint (WB3c: is_ext_pict)
//   ncat   — first non-ignorable after cur (WB6/WB7b/WB12), 0 если конец
//   ri_run — RI parity counter
fn wb_is_break_s(p_imm int, pcat int, ppcat int, c int, cur_cp int, ncat int, ri_run int) -> bool {
    if p_imm == WB_CR && c == WB_LF {
        false                                                       // WB3
    } else if p_imm == WB_NEWLINE || p_imm == WB_CR || p_imm == WB_LF {
        true                                                        // WB3a
    } else if c == WB_NEWLINE || c == WB_CR || c == WB_LF {
        true                                                        // WB3b
    } else if p_imm == WB_ZWJ && is_ext_pict(cur_cp) {
        false                                                       // WB3c
    } else if p_imm == WB_WSEGSPACE && c == WB_WSEGSPACE {
        false                                                       // WB3d
    } else if wb_ignorable(c) {
        false                                                       // WB4
    } else if wb_ahletter(pcat) && wb_ahletter(c) {
        false                                                       // WB5
    } else if wb_ahletter(pcat) && (c == WB_MIDLETTER || wb_midnumletq(c)) && wb_ahletter(ncat) {
        false                                                       // WB6
    } else if (pcat == WB_MIDLETTER || wb_midnumletq(pcat)) && wb_ahletter(c) && wb_ahletter(ppcat) {
        false                                                       // WB7
    } else if pcat == WB_HEBREW && c == WB_SQ {
        false                                                       // WB7a
    } else if pcat == WB_HEBREW && c == WB_DQ && ncat == WB_HEBREW {
        false                                                       // WB7b
    } else if ppcat == WB_HEBREW && pcat == WB_DQ && c == WB_HEBREW {
        false                                                       // WB7c
    } else if pcat == WB_NUMERIC && c == WB_NUMERIC {
        false                                                       // WB8
    } else if wb_ahletter(pcat) && c == WB_NUMERIC {
        false                                                       // WB9
    } else if pcat == WB_NUMERIC && wb_ahletter(c) {
        false                                                       // WB10
    } else if (pcat == WB_MIDNUM || wb_midnumletq(pcat)) && c == WB_NUMERIC && ppcat == WB_NUMERIC {
        false                                                       // WB11
    } else if pcat == WB_NUMERIC && (c == WB_MIDNUM || wb_midnumletq(c)) && ncat == WB_NUMERIC {
        false                                                       // WB12
    } else if pcat == WB_KATAKANA && c == WB_KATAKANA {
        false                                                       // WB13
    } else if (wb_ahletter(pcat) || pcat == WB_NUMERIC || pcat == WB_KATAKANA || pcat == WB_EXTENDNUMLET) && c == WB_EXTENDNUMLET {
        false                                                       // WB13a
    } else if pcat == WB_EXTENDNUMLET && (wb_ahletter(c) || c == WB_NUMERIC || c == WB_KATAKANA) {
        false                                                       // WB13b
    } else if pcat == WB_RI && c == WB_RI && (ri_run % 2 == 1) {
        false                                                       // WB15/16
    } else {
        true                                                        // WB999
    }
}
```

### 3.3 Lookahead helper

```nova
// Первый non-ignorable WB-category начиная с байт-офсета `from`. 0 если конец строки.
fn scan_next_eff_cat(s str, from int, n int) -> int {
    mut j = from
    while j < n {
        ro (cp, step) = decode_at(s, j)   // decode_at из graphemes.nv (peer)
        ro cat = wbcat(cp)
        if !wb_ignorable(cat) { return cat }
        j = j + step
    }
    0
}
```

### 3.4 Методы `WordsIter`

```nova
export fn WordsIter @iter() -> WordsIter => @

export fn WordsIter @is_empty() -> bool => @pos >= @n

// O(n) — нет precomputed bounds, документируем явно.
export fn WordsIter @count() -> int {
    mut tmp = @
    mut cnt = 0
    while tmp.next() != None { cnt = cnt + 1 }
    cnt
}

export fn WordsIter mut @next() -> Option[str] {
    if @pos >= @n { return None }
    ro start = @pos
    // Первый cp сегмента — всегда включается, break не проверяем.
    ro (cp0, step0) = decode_at(@buf, @pos)
    ro cat0 = wbcat(cp0)
    @_advance(cat0)
    @pos = @pos + step0
    while @pos < @n {
        ro (cp, step) = decode_at(@buf, @pos)
        ro cat = wbcat(cp)
        ro ncat = scan_next_eff_cat(@buf, @pos + step, @n)
        if wb_is_break_s(@prev_imm_cat, @prev_eff_cat, @prev_prev_eff_cat,
                          cat, cp, ncat, @ri_run) {
            return Some(@buf[start..@pos])
        }
        @_advance(cat)
        @pos = @pos + step
    }
    Some(@buf[start..@n])
}

// Module-private: обновить state после обработки cp с категорией cat.
fn WordsIter mut @_advance(cat int) {
    @prev_imm_cat = cat
    @ri_run = wb_ri_step(cat, @ri_run)
    if !wb_ignorable(cat) {
        @prev_prev_eff_cat = @prev_eff_cat
        @prev_eff_cat = cat
    }
}
```

## Ф.6 — `decode_at` dedup в `runtime.string`

**Цель:** устранить три копии UTF-8 decode в `std/runtime/string/chars.nv`.

### 6.1 Добавить module-private helper

В `std/runtime/string/chars.nv` после `fn CharsIter.new(...)`:

```nova
// Module-private UTF-8 decode: (codepoint, byte_step). R-UTF8: str всегда валидный.
// Сигнатура отличается от graphemes.nv peer: принимает уже вычисленный bytes+n
// (избегает повторного .as_bytes()/.byte_len() в tight loop).
fn decode_utf8(bytes ro []u8, i int, n int) -> (int, int) {
    ro b = bytes[i] as int
    if b < 0x80 {
        (b, 1)
    } else if ((b & 0xE0) == 0xC0) && (i + 1 < n) {
        (((b & 0x1F) << 6) | (bytes[i+1] as int & 0x3F), 2)
    } else if ((b & 0xF0) == 0xE0) && (i + 2 < n) {
        (((b & 0x0F) << 12) | ((bytes[i+1] as int & 0x3F) << 6) | (bytes[i+2] as int & 0x3F), 3)
    } else if ((b & 0xF8) == 0xF0) && (i + 3 < n) {
        (((b & 0x07) << 18) | ((bytes[i+1] as int & 0x3F) << 12)
            | ((bytes[i+2] as int & 0x3F) << 6) | (bytes[i+3] as int & 0x3F), 4)
    } else {
        (b, 1)
    }
}
```

### 6.2 Переписать `@next()`

```nova
export fn CharsIter mut @next() -> Option[char] {
    ro (bytes, n) = (@buf.as_bytes(), @buf.byte_len())
    if @pos >= n { return None }
    ro (cp, step) = decode_utf8(bytes, @pos, n)
    @pos = @pos + step
    Some(cp_to_char(cp))
}
```

### 6.3 Переписать `@nth()`

```nova
export fn CharsIter @nth(idx int) -> Option[char] {
    if idx < 0 { return None }
    ro (bytes, n) = (@buf.as_bytes(), @buf.byte_len())
    mut cp_idx = 0
    mut i = @pos
    while i < n {
        ro (cp, step) = decode_utf8(bytes, i, n)
        if cp_idx == idx { return Some(cp_to_char(cp)) }
        cp_idx = cp_idx + 1
        i = i + step
    }
    None
}
```

### 6.4 Переписать `@to_chars()` в `core.nv`

Найти `fn str @to_chars()` в `std/runtime/string/core.nv`:

```nova
export fn str @to_chars() -> []char {
    ro (bytes, n) = (@as_bytes(), @byte_len())
    mut out = []char.with_capacity(n)
    mut i = 0
    while i < n {
        ro (cp, step) = decode_utf8(bytes, i, n)
        out.push(cp_to_char(cp))
        i = i + step
    }
    out
}
```

## Ф.7 — Perf: cps_to_str + chains + bulk copy

### П1. `cps_to_str` — N temp-str аллокаций → append(char)

**Файл:** `std/unicode/cp_utils.nv` (перенесён из normalize.nv в Ф.4)

Найти функцию `cps_to_str(cps Vec[u32]) -> str`. Текущее тело создаёт N временных строк:
```nova

```

Заменить (StringBuilder @append(c char) кодирует UTF-8 напрямую):
```nova
fn cps_to_str(cps Vec[u32]) -> str {
    mut sb = StringBuilder.new()
    for cp in cps {
        match char.try_from(cp as int) {
            Ok(c)  => sb.append(c)
            Err(_) => sb.append('\u{FFFD}')
        }
    }
    sb.into()
}
```

Тот же паттерн в `std/encoding/utf16.nv` в `from_utf16` — аналогично исправить.

### П2. Method chains

**`std/runtime/string/core.nv` — `@to_bytes()`:**

### П3. `@to_ascii_lower/upper` — bulk copy + in-place patch

**Файл:** `std/runtime/string/transform.nv`

```nova
export fn str @to_ascii_lower() -> str {
    ro (bytes, n) = (@as_bytes(), @byte_len())
    mut out = []u8.with_capacity(n).append(bytes)   // 1 RawMem.copy
    mut i = 0
    while i < n {

        if b >= 65 && b <= 90 { out[i] = (b + 32) as u8 }   // A-Z → a-z
        i = i + 1
    }

        ro b = out[i] as int
        if b >= 97 && b <= 122 { out[i] = (b - 32) as u8 }   // a-z → A-Z
        i = i + 1
    }
    str.from_bytes_unchecked_steal(out)
}
```
