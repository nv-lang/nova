## Цель
Заморозить **ABI shadow-frame** (§7.6 ревизия: плоский `void* slot[N]` сохраняется;
interior — object-start lookup на mark, БЕЗ base/offset в кадре). Спроектировать:
- **object-start lookup-структуру** (block / object-start table) — по interior-указателю
  найти начало владеющего GC-буфера; тотальна (не-GC адрес → «не найдено → skip»);
- определение **safe-point** (call/alloc/loop-back-edge);
- протокол **swap вершины** `nova_shadow_top` на fiber-switch (рядом со swap'ом стека);
- взаимодействие с **`blocking{}`-offload** (вершина в own root-frame offload-потока);
- дисциплину **pop** на error/panic/defer (единый epilogue + unwinder-reset, §7.5 #7);
- **unified roots registry** (реестр всех живых вершин: per-fiber + per-worker/offload).

**Закрыть оставшиеся prerequisites §7.6:**
- **H5** — захват замыканий по указателю в кадры: выбрать **by-value boxing** (Go-style,
  делает Q10 истинным) vs `kind=interior-in-frame + запрет move`. Ф.0-блокер.
- **H3** — `blocking{}`-FFI статически → **Tier-2** (pin/copy); базы в own root-frame.
- *(H4 — may-GC дефолт MayGC top — ✅ уже закрыт [Plan 144.0](144.0-may-gc-effect-analysis.md),
  [D273](../../spec/decisions/06-concurrency.md#d273). H1 закрыт ревизией §7.6.)*

## Acceptance criteria
- Design-doc + spec D-блок (D2xx) + open-questions.
- object-start lookup + реестр вершин **описаны**.
- **H3 и H5 разрешены** (H4-слайс ✅ закрыт Plan 144.0).
- **non-moving-first зафиксировано**; base+offset заложен в ABI как moving-only задел
  (не запирать moving навсегда — ловушка Julia), но коллектор строить под non-moving.
- Без кода.
- «без упрощений как для прода» — обязательный критерий.
