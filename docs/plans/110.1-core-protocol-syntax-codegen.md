## Фазы

### Ф.1 — Core: protocol + parser + checker

Все sub-фазы Plan 110 §Ф.1 (Ф.1.1 — Ф.1.8).

- **Ф.1.1** Prelude: `Consumable[E]` в `std/prelude/protocols.nv` +
  `WithExitTimeout` (отдельный protocol) + `ScopeOutcome` sum-type
  в `std/prelude/runtime.nv` (или новый `cleanup.nv` submodule).
- **Ф.1.2** Никаких изменений `never` keyword (уже lowercase per Plan 76).
- **Ф.1.3** Parser lookahead на `{` после `EXPR` в `consume IDENT = EXPR`.
  Грамматика: `stmt → consume IDENT (':' TYPE)? '=' EXPR ('{' block_body '}')?`.
- **Ф.1.4** AST: добавить `Stmt::ConsumeScope { binding, init_expr, body }`
  в `compiler-codegen/src/ast/mod.rs`.
- **Ф.1.5** Type-checker D188 R1-R6 + D196:
  - `init: Consumable[E]` structural match;
  - вывод `E`;
  - `Consumable[Never]` permits caller без `Fail[E]`;
  - init форма constraints (Result/Option unwrap, conditional, method chain).
- **Ф.1.6** D162 simplified: `consume {}` exhaustive by construction →
  no D162-check для scope-bindings; rule сохраняется для raw `consume + defer`.
- **Ф.1.7** Generic bound `[T Consumable[E]]` + Never special case.
- **Ф.1.8** Tests T1.1–T1.8.

### Ф.2 — Codegen + runtime fundamentals

- **Ф.2.1** Codegen desugaring (см. D188 §«Desugaring»).
- **Ф.2.2** Runtime:
  - `nv_consume_enter(timeout_ns) -> nv_consume_scope_t*`;
  - `nv_consume_exit(scope, outcome_kind, error_ptr)`;
  - `nv_consume_drop_scope(scope)` для memory cleanup.
- **Ф.2.3** Exactly-once invariant — runtime counter; panic при ≥ 2.
- **Ф.2.4** Partial-construction safety — `init succeeded` flag в codegen.
- **Ф.2.5** LIFO composition — scope-stack per-fiber.
- **Ф.2.6** Mixed `consume {}` + `defer` LIFO — shared stack.
- **Ф.2.7** `nv_resolve_exit_timeout(typeid, has_with_exit_timeout)` —
  единая runtime fn через vtable lookup (не per-callsite codegen).
- **Ф.2.8** Hot-path opt (D194 §perf): `Consumable[Never]` + no
  `WithExitTimeout` → elide shield/timeout/outcome.
- **Ф.2.9** Init type constraints (D196).
- **Ф.2.10** ~~`outcome.failure_as[T]()` helper~~ — **отвергнут** (D190);
  используется прямой `if err is T` (D85).
- **Ф.2.11** Generic `[T Consumable[Never]]` — type-checker снимает `Fail[E]`.
- **Ф.2.12** Cleanup re-entrance (D197) basic — без cancel-shield (тот
  делается в Plan 110.2).
- **Ф.2.13** Tests T2.1–T2.12 (без cancel/async — те в Plan 110.2).
