# Plan 153.3.1 — pdqsort в @sort_unstable* ✅ CLOSED 2026-06-18

Заменяет heapsort на pdqsort (pattern-defeating quicksort) в
`@sort_unstable` / `@sort_unstable_by` / `@sort_unstable_by_key`.

## Алгоритм (итеративный, без рекурсии)

- n ≤ 1  → return immediately
- n ≤ 16 → insertion sort (cache-warm, branch-free)
- depth_limit = 2·ilog2(n) исчерпан → heapsort fallback (O(n log n) гарантия)
- иначе: median-of-3 pivot (→ index hi-1) + Lomuto partition + Vec[int] work-stack

Стек хранит пары `(lo, hi)` как consecutive int's (lo at i*2, hi at i*2+1).

## Файлы

- `std/collections/vec/sort.nv` — добавлены `@_ins_sort_range`, `_pdq_ilog2`, `@_pdqsort`;
  `@sort_unstable*` теперь вызывают `@_pdqsort` вместо `@_heapsort`.
