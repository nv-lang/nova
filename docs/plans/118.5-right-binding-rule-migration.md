## –Я–Њ—З–µ–Љ—Г

–Ґ–µ–Ї—Г—Й–∞—П –≥—А–∞–Љ–Љ–∞—В–Є–Ї–∞ —В–Є–њ–Њ–≤ –Є–Љ–µ–µ—В **–і–≤–∞ –њ—А–∞–≤–Є–ї–∞** –Њ–і–љ–Њ–≤—А–µ–Љ–µ–љ–љ–Њ:
- `ro T` parses –Ї–∞–Ї right-binding (`Readonly(T)` —З–µ—А–µ–Ј recursive `parse_type`)
- `*ro T` / `*mut T` / `*unsafe T` parses –Ї–∞–Ї inline-modifier (modifier –≤–љ—Г—В—А–Є
  pointer constructor)

Inconsistency –і–Є—Б–Ї–Њ–Љ—Д–Њ—А—В–љ–∞ –і–ї—П users (intuition –і–Є–Ї—В—Г–µ—В –Њ–і–љ–Њ –њ—А–∞–≤–Є–ї–Њ –і–ї—П
–≤—Б–µ—Е modifiers) –Є –Ј–∞–Ї—А—Л–≤–∞–µ—В orthogonal expressiveness:
- `unsafe * T` (unsafe pointer) вЖФ `* unsafe T` (valid ptr to uninit T) вАФ
  –љ–µ–≤–Њ–Ј–Љ–Њ–ґ–љ–Њ express –њ–Њ–і current inline-modifier scheme

Migration cost –ї–Є–љ–µ–є–љ–Њ —Б FFI footprint вАФ fix –Э–Ю–Т–Ђ–Х deferred plans (118.1
–§.2-–§.4 / 118.2 / 118.3) –њ–Њ–і –љ–Њ–≤—Л–є syntax –і–Њ —В–Њ–≥–Њ –Ї–∞–Ї accumulate'–љ—Г—В—Б—П
fixtures –њ–Њ–і —Б—В–∞—А—Л–є.
