1. **U.4.3 typedef-drop**: удаление :36334 ДО ФАЗА 3 → пропадёт ранняя return-mono цепочек `obj.vmap(f).collect()` → NULL-stub/segfault/CC-FAIL. Mitigation: ФАЗА 3 строго ПЕРЕД ФАЗА 6.

3. **SelfAccess mono-subst gap #2**: channel resolved_type_to_c(scope["@"]) ненадёжен в mono-теле (current_type_subst timing) — источник ОБЯЗАН остаться var_types["nova_self"]. Mitigation: ФАЗА 4B inline, не channel.
4. **fn_ret_by_span carrier-смена**: широкий blast-radius, риск регресса extern Gap-B (try_start_won→bool). Верифицировать NOVA_U45A_XCHECK.
