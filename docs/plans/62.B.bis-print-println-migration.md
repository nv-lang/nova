## Проблема (verified state, 2026-05-18)

### 1. Hardcoded variadic + type-polymorphic dispatch

`compiler-codegen/src/codegen/emit_c.rs:11330-11334`:
```rust
if let ExprKind::Ident(name) = &func.kind {
    if name == "println" || name == "print" {
        return self.emit_println(args, name == "println");
    }
```

`emit_println` (`emit_c.rs:13969-13997`) — per-arg variadic emit:
```rust
fn emit_println(&mut self, args: &[CallArg], newline: bool) -> Result<String, String> {
    // ... per-arg loop, each emits make_print_call() ...
    if call_arg.is_spread() {
        return Err("spread (...) в println/print пока не поддержан".into());
    }
    // ...
}
```

`infer_print_helper` (`emit_c.rs:14006-14082`) — manual type heuristics:
- `IntLit` → `nova_print_int`, `StrLit` → `nova_print_str`, `BoolLit` → `nova_print_bool`, `FloatLit` → `nova_print_f64`
- `Ident` → lookup `var_types` HashMap, fallback `nova_print_int`
- `Member` → record-schema lookup, fallback `nova_print_int`
- `Call` → fragmented logic for `obj.method()` (str methods hardcoded list), `func(args)` (lookup `fn_ret_<name>`)
- Default everywhere → `nova_print_int` (silent garbage для unrecognized return types — corollary of Plan 67 bug).

## Что НЕ делаем (rejected alternatives)

| Alternative | Почему отвергнут |
|---|---|
| **Single-arg `external fn print(s str) -> ()`** | Breaking change для 80+ multi-arg call-sites. Нарушает D26 canonical signature. |
| **Approach (b): Display protocol + per-arg fmt(sb) → single str** | Требует Display impl для каждого primitive (только user-types через D73). Bootstrap primitives печатаются через hardcoded helpers — нет formal `int.@fmt(sb)`. Полная Display-pipelining requires расширения primitive method tables в codegen (2-3 dev-days). StringBuilder allocation per println call — perf regression. |
| **Approach (c): Hybrid — multiple arity overloads** | 16 declarations × 2. Verbose. Real call-sites имеют до 5 args. Не canonical D26 sig. |
| **Approach (d): Interpolation-only migration** | 80+ call-sites mass rewrite. 60 файлов. Очень invasive. Не помогает spec'у. |
| **Approach (e): Remove print/println from prelude** | Spec D26 явно перечисляет. Breaking spec change. |
| **Hardcoded codegen + declaration без dispatch change** | Уже модель для panic/exit/assert/debug_assert в 62.B partial. **Это и есть recommended approach (a)** ниже. |
| **Removing hardcoded special-case в emit_c.rs immediately** | Требует `nova_print_any` runtime helper + boxing pipeline. Slow и invasive. |

## Решение — архитектурные правила

### Approach (a): Formal `[]any` variadic declaration + preserved special-case codegen

**Архитектурный принцип:** D26 spec's canonical signature `fn print(...items []any) Io -> ()` становится **source of truth для type-checker'а и `nova doc`**; codegen продолжает использовать `emit_println` special-case как **bootstrap implementation**, эмитирующий per-arg `nova_print_<type>` helpers. Mirror'ит pattern panic/exit/assert/debug_assert из 62.B partial.

**Почему это правильно для bootstrap'а:**

1. **D69 variadic infrastructure уже работает.** `external fn print(...items []any)` парсится, type-checker'у понятно.
2. **`any` уже builtin primitive.**
3. **Codegen special-case stays** — minimal change, zero regression risk на 80+ call-sites.
4. **Spec/doc drift closed** — `nova doc std/prelude/runtime.nv` показывает canonical sig.
5. **Symmetry с panic/exit/assert/debug_assert** в 62.B partial.

### Spec D26 alignment

```nova
/// Print all arguments to stdout, no separator, no trailing newline.
#stable(since = "0.1")
external fn print(...items []any) -> ()

#stable(since = "0.1")
external fn println(...items []any) -> ()
```

Notably:
- **No `Io` effect в bootstrap** — `Io` не declared в `std/prelude/effects.nv`. Consistent с panic/exit/assert/debug_assert.
- `Never` effect не нужен — return type `()`.
- Generic params отсутствуют — `[]any` accepts heterogeneous args без `T` parameter.

### Codegen impact — minimal

1. **`emit_c.rs:11330-11334`** — остаётся без изменений. Special-case intercept fires для AST Ident `println`/`print` (после HashSet shrink identifier name остаётся unqualified).
2. **`emit_println`** — остаётся без изменений. Variadic dispatch per arg сохраняется. Spread продолжает не работать (same DEFER).
3. **`infer_print_helper`** — **refactor per Plan 67 AD1** (Ф.0). Replace manual heuristics с `infer_expr_c_type`-based dispatch.
4. **Type-checker `[]any` variadic acceptance** — special-case intercept fires **before** variadic routing, synthesized array никогда не строится.
5. **Generic `external_registry`** — no change.

### Stratification

`std/prelude/runtime.nv` имеет **zero imports**. Добавление 2 declarations с `[]any` — zero new imports (`any` builtin, `[]` array built-in).
