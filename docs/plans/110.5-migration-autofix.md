# Plan 110.5: Migration deprecation + auto-fix tool (Ф.9)

> **Создан 2026-05-31.** Sub-plan Plan 110 §«Возможный split». Реализует
> Ф.9 — `nova fix --simplify-cleanup` auto-migration tool + парсер reject'ы
> старых форм + D90 §7 cancel-as-failure codegen.
>
> **Цель:** auto-migrate ≥ 95% существующих `okdefer`/`errdefer`/
> `defer |result|` usage'ов в `consume X = ... { ... }` или `defer + flag`
> патерны (3 правила из D189). После tool run — удалить парсер-поддержку
> старых форм.
>
> **D-блоки:** активирует D189 (parser reject post-removal), D90 §7 amend
> (cancel/interrupt → Failure(CancelError)).
>
> **Acceptance:** A11, A12, A22 (D90 §7 amend).
>
> **Зависимости:** Plan 110.1 (compiler pipeline + new ConsumeScope parsing
> path); Plan 110.4 (CancelError prelude type).

## Фазы

### Ф.9 — Migration deprecation + auto-fix

- **Ф.9.1** Parser принимает старые формы + emit warnings (transition window).
- **Ф.9.2** Auto-migration tool `nova fix --simplify-cleanup`:
  - Pattern 1: `consume tx = ...; errdefer { rollback }; okdefer { commit }`
    → `consume tx = ... { ... }` (предполагает Transaction Consumable impl).
  - Pattern 2: bare `errdefer { x }` → `mut done = false; defer { if !done
    { x } }; ...; done = true`.
  - Pattern 3: `defer |result|` → `consume X = ... { ... }` или
    `with Cleanup = h { ... }`.
- **Ф.9.3** Migration существующих фикстур `nova_tests/plan100_4_*` (см.
  migration audit Plan 110 Ф.0.2 — 42 файла, 90 occurrences).
- **Ф.9.4** D90 §7 codegen: cancel/interrupt в body → `Failure(CancelError)`.
- **Ф.9.5** Удаление `DeferWithResult` AST node + `DeferResult[T,E]` prelude.
- **Ф.9.6** Удаление `errdefer`/`okdefer`/`defer |r|` parser support
  после Ф.9.2 tool run + Ф.9.3 fixture migration.
- **Ф.9.7** Tests T9.1–T9.5.

## Tool deliverable

`nova fix --simplify-cleanup [PATH]`:
- по умолчанию full workspace migration;
- `--dry-run` — preview changes без записи;
- `--report` — JSON report с counters (patterns matched, files touched).

## Acceptance

A11 (deprecation + auto-fix), A12 (D90 §7 amend cancel→Failure).

## Оценка

~1-2 dev-day. Sequential. Tool — Rust binary в `nova-cli`.
