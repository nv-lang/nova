## 1. Контекст

V7 / V6.3 iterative closure runs `for _ in 0..iter_limit` over the
call graph. Defaults к 10 iterations — converges для typical Nova
modules в ≤3. Adversarial 8+ deep mutual-recursion may terminate
prematurely с silent under-propagation. V7.3 replaces it с Tarjan
SCC + reverse-topological exact propagation.

## 2. Изменения

### 2.1 field_cache.rs

New private fns:
- `propagate_via_scc(direct, callees)` — entry point.
- `tarjan_scc(adj) -> Vec<Vec<usize>>` — iterative Tarjan.

`build_write_set_registry` / `build_read_set_registry` теперь:
- Check `NOVA_FC_LEGACY_ITERATIVE_CLOSURE` env var.
- If `=1` → V7 iterative loop (forensic fallback).
- Иначе → `propagate_via_scc(...)` (default V7.3 path).

### 2.2 Algorithm details

1. Node indexing: union direct.keys ∪ callees.keys ∪ callees.values.
2. Adjacency list built from `callees` map.
3. `tarjan_scc` — iterative work-stack DFS (avoids Rust stack on
   deep graphs). Returns SCCs leaves-first.
4. SCC visit: pool = direct ∪ outbound-edge SCC sets. Assign к все
   members.
