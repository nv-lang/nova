### 2.2 Token emission algorithm

1. Run best-effort pipeline (parse → check → desugar → infer →
   normalize → analyze_module).
2. Build per-fn cached-field set (ro/mut/licm/chain-root).
3. Walk source bytes; for each `@<name>` token, find enclosing fn
   and check membership.
4. Sort raw tokens by (line, col); delta-encode per LSP spec.
