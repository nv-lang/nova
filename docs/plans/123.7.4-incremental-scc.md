## 1. Контекст

V7.3 emits exact Tarjan SCC + reverse-topological propagation per
`cache_module` invocation. ~1ms на typical Nova module (300+ methods).
Реалистичные workloads (LSP rechecks с debouncer, IDE batch passes,
build-cache hits) репитят `cache_module` на identical modules и
платят полную SCC-цену каждый раз — pure waste.

V7.4 добавляет process-level memoization: fingerprint входной граф,
кешируется propagated `direct` map по fingerprint, restore на hit.
O(1) для repeated identical inputs.

## 2. Изменения

### 2.1 `compiler-codegen/src/field_cache.rs` (~190 LOC core + 240 tests)

**New public types:**

```rust
pub struct ScCache {
    last_fingerprint: u64,
    last_result: HashMap<(String, String), HashSet<String>>,
    has_entry: bool,
    pub hits: u64,
    pub misses: u64,
}
impl ScCache { pub fn reset(&mut self); }
```

**New static singletons:**

```rust
static WRITE_SET_SCC_CACHE: OnceLock<Mutex<ScCache>>;
static READ_SET_SCC_CACHE:  OnceLock<Mutex<ScCache>>;
fn write_set_scc_cache() -> &'static Mutex<ScCache>;
fn read_set_scc_cache()  -> &'static Mutex<ScCache>;
```

**New public API:**

```rust
pub fn scc_cache_enabled() -> bool;
pub fn scc_cache_stats() -> (u64, u64, u64, u64);
pub fn reset_scc_caches();
```

**Core functions:**

- `compute_scc_fingerprint(direct, callees) -> u64` — canonical
  BTreeMap-sorted hash via DefaultHasher (SipHash-1-3). Reserves 0
  as sentinel; biases empty-graph к 1.
- `propagate_via_scc_cached(direct, callees, cache_cell)` — wrapper
  around V7.3 `propagate_via_scc`. No-op passthrough when cache
  disabled.

**Wired in:**

- `build_write_set_registry` → `propagate_via_scc_cached(..,
  write_set_scc_cache())`.
- `build_read_set_registry` → `propagate_via_scc_cached(..,
  read_set_scc_cache())`.

### 2.2 Unit tests (10 V7.4 tests в `field_cache::tests`)

**Test infra:**
- `with_scc_env(enabled, body)` — RAII-style env + cache state guard,
  serializes test execution через static `Mutex<()>` (cargo runs
  tests in same-process threads; env-var mutation isn't thread-safe).
- `sample_graph()` — small T.{a, b} graph fixture.

**Positive (6):**
- `v74_cache_hit_on_identical_input` — second identical call hits.
- `v74_fingerprint_stable_across_hashmap_order` — determinism.
- `v74_hits_misses_counters_track_correctly` — 3 hits + 1 miss
  after 4 calls.
- `v74_changed_graph_triggers_miss` — added node + edge → fingerprint
  mismatches → miss.
- `v74_write_and_read_caches_isolated` — separate slots verified.
- `v74_reset_clears_cache_state` — counters zeroed, next call misses.

**Negative (4):**
- `v74_cache_disabled_by_default` — без env → counters stay (0, 0).
- `v74_empty_graph_fingerprint_nonzero` — sentinel reservation honored.
- `v74_distinct_graphs_distinct_fingerprints` — collision-resistance
  на realistic 2-step modifications.
- `v74_cache_hit_preserves_v73_semantics` — hit path bitwise-equal
  к miss path bitwise-equal к V7.3 baseline.

  Future: detect LSP-server runtime и enable transparently.
  Followup V7.4.3.
