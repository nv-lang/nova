## 1. Проблема

Nova позиционируется как **backend / CLI** язык (Plan 18 P0 — backend
focus). Без production-grade async TCP/UDP — все Nova programs не могут
быть servers, network clients, или integrate с remote services. Это
fundamental gap.

**Текущее состояние:**
- Plan 22 ✅ — libuv integration baseline (timer, fs sleep).
- Plan 18 P0 — stated as "unblocked" в 42.12 closure note, но **actual
  net/socket stdlib НЕ landed**.
- Нет `std/net/`, нет `std/socket/`, нет HTTP layer (P1).

**Industry baseline:**
- Go: `net/http`, `net/tcp`, `net/udp` — comprehensive std.
- Tokio: `tokio::net::{TcpListener, TcpStream, UdpSocket}`.
- Kotlin: `ktor.network.*`.
- Node: `net.createServer()`, `dgram.createSocket()`.

**Nova: должен быть.**

## 2. Scope V1

Минимальный production-grade subset:

```
std/net/
├── tcp.nv         # TcpListener + TcpStream + TcpClient
├── udp.nv         # UdpSocket
├── addr.nv       # SocketAddr (IPv4/IPv6)
├── error.nv     # NetError typed errors
└── _module.nv   # module-level forbid attrs
```

**Out of scope V1:**
- TLS/SSL (V2 → отдельный план через rustls FFI).
- HTTP layer (V2 → Plan 18 P1).
- Unix domain sockets (V2).
- Async DNS resolver — V1 uses sync getaddrinfo через uv_getaddrinfo.

## 3. Дизайн

### 3.1 API (sketch)

```nv
// std/net/tcp.nv

use std::net::addr::SocketAddr
use std::net::error::NetError

record TcpListener {
    inner: *opaque  // uv_tcp_t handle
}

record TcpStream {
    inner: *opaque  // uv_tcp_t handle
    local: SocketAddr
    peer: SocketAddr
}

impl TcpListener {
    pub fn bind(addr: SocketAddr) -> Result[TcpListener, NetError] [IO, Blocking]
    pub fn accept(self) -> Result[TcpStream, NetError] [IO, Blocking]
    pub fn close(self) [IO]
}

impl TcpStream {
    pub fn connect(addr: SocketAddr) -> Result[TcpStream, NetError] [IO, Blocking]
    pub fn read(self, buf: ReadBuffer) -> Result[int, NetError] [IO, Blocking]
    pub fn write(self, data: String) -> Result[int, NetError] [IO, Blocking]
    pub fn close(self) [IO]
    pub fn shutdown(self, how: ShutdownHow) [IO]
}

enum ShutdownHow { Read, Write, Both }

enum NetError {
    ConnectionRefused
    AddressInUse
    NotFound
    Closed
    Cancelled
    IoError(String)
    InvalidAddr(String)
}
```

```nv
// std/net/udp.nv

record UdpSocket {
    inner: *opaque  // uv_udp_t
}

impl UdpSocket {
    pub fn bind(addr: SocketAddr) -> Result[UdpSocket, NetError] [IO, Blocking]
    pub fn send_to(self, data: String, addr: SocketAddr) -> Result[int, NetError] [IO]
    pub fn recv_from(self, buf: ReadBuffer) -> Result[Tuple[int, SocketAddr], NetError] [IO, Blocking]
    pub fn close(self) [IO]
}
```

### 3.2 Cancel integration

Все blocking operations интегрируются с supervised cancel:

```nv
supervised(cancel: tok) {
    let stream = TcpStream.connect(addr)  // cancels через tok
}
```

Implementation: register handle с `_nova_active_scope.sched_state`
pending_handle list + `NovaStopMode::ASYNC` (uv_close на cancel).
Аналогично Plan 22 sleep integration.

### 3.3 Capability + effect rules

- All net ops carry `IO + Blocking` effects.
- Module `_module.nv` опционально опционально декларирует `#forbid(IO)` —
  disable net в realtime-critical modules.
- `unrestricted` capability checked при FFI binding.

### 3.4 Implementation (codegen + runtime)

**Files новые:**
- `compiler-codegen/nova_rt/net.h` — declarations.
- `compiler-codegen/nova_rt/net.c` — libuv wrappers с park/wake.
- `compiler-codegen/src/codegen/emit_net.rs` — codegen routing для
  `std/net/*` functions через `external fn`.

Pattern follows Plan 22 sleep integration:
1. Caller fiber parks via `nova_sched_park_until` (D93).
2. libuv `uv_tcp_*` operation issued с callback.
3. Callback wakes fiber + delivers result.
4. Cancel → `uv_close` через stop_cb (NOVA_STOP_ASYNC).

### 3.5 Address parsing

```nv
// std/net/addr.nv

record SocketAddr {
    host: String  // "192.168.1.1" либо "::1" либо "example.com"
    port: u16
    family: AddrFamily
}

enum AddrFamily { V4, V6 }

impl SocketAddr {
    pub fn parse(s: String) -> Result[SocketAddr, NetError]
    pub fn v4(a: u8, b: u8, c: u8, d: u8, port: u16) -> SocketAddr
    pub fn loopback(port: u16) -> SocketAddr
}
```

DNS resolution V1 — sync через uv_getaddrinfo (blocking effect required).

## 4. Тесты

### Positive (новые):

**`tcp_echo_server_test.nv`** — full echo round-trip:
```nv
use std::net::*

test "TCP echo client + server" {
    let listener = TcpListener.bind(SocketAddr.loopback(0))?
    let server_port = listener.local_port()

    supervised {
        // server fiber
        spawn {
            let stream = listener.accept()?
            let data = stream.read(buf)?
            stream.write(data)?
            stream.close()
        }

        // client fiber
        spawn {
            let stream = TcpStream.connect(SocketAddr.loopback(server_port))?
            stream.write("hello")?
            let echo = stream.read(buf)?
            assert(echo == "hello")
            stream.close()
        }
    }
}
```

**`udp_ping_pong_test.nv`** — UDP datagram exchange.

**`tcp_multi_connection_test.nv`** — 10³ concurrent connections, verify
no leak.

**`tcp_listener_cancel_test.nv`** — cancel listener.accept() через token.

**`tcp_with_deadline_test.nv`** — combination с Plan 83.11 deadline.

### Negative (новые):

**`tcp_connect_refused_test.nv`**:
```nv
// EXPECT_RUNTIME_ERROR

test "TCP connect to unused port — typed error not panic" {
    let r = TcpStream.connect(SocketAddr.loopback(1))
    match r {
        Err(NetError.ConnectionRefused) => {}
        _ => assert(false)
    }
}
```

**`tcp_listener_bind_error_test.nv`** — port in use → typed error.

**`tcp_read_after_close_test.nv`** — read after peer close → EOF marker.

**`tcp_write_to_closed_test.nv`** — write after close → typed error.

**`udp_oversized_packet_test.nv`** — datagram > MTU → typed error.

**`tcp_socket_leak_under_cancel_test.nv`**:
```nv
// Verify socket descriptors closed under cancel.
test "TCP sockets closed under cancel — no leak" {
    use std::runtime::cancel
    let tok = cancel.token()
    supervised(cancel: tok) {
        spawn {
            let stream = TcpStream.connect(addr)?
            // long read blocked
            stream.read(buf)?
        }
        Time.sleep(10ms)
        tok.cancel()
    }
    // Verify socket освобождён (libuv close_cb fired).
}
```

**`tcp_invalid_addr_compile_error.nv`** — Compile-time addr literal
validation (if implemented as compile-time check).
