## TL;DR

```nova
// (1) loop-var-mut
for x in arrs { x.push(1) }           // ✗ E_LOCAL_NOT_MUT (Plan 108.2)
for mut x in arrs { x.push(1) }       // ✓ Plan 108.3

// (2) pattern-binding mut
let (a, b) = pair                     // оба immutable
let (mut a, b) = pair                 // a mutable, b immutable
a.push(1)                             // ✓
b.push(1)                             // ✗ E_LOCAL_NOT_MUT

// (3) residual migrations

let mut writer_done = AtomicBool.new(false)
```
