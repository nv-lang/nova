## –Ч–∞—З–µ–Љ

–Ґ–µ–Ї—Г—Й–Є–є `Once` baseline (`std/runtime/sync.nv:172-214`) вАФ –і–≤—Г—Е-—И–∞–≥–Њ–≤—Л–є
`run()/done()` pattern: –њ–Њ–ї—М–Ј–Њ–≤–∞—В–µ–ї—М –і–Њ–ї–ґ–µ–љ –њ–Њ–Љ–љ–Є—В—М –≤—Л–Ј–≤–∞—В—М `done()`,
–Є–љ–∞—З–µ –≤—Б–µ waiters –љ–∞–≤—Б–µ–≥–і–∞ –Ј–∞–±–ї–Њ–Ї–Є—А–Њ–≤–∞–љ—Л. –≠—В–Њ **fragile API** (M8):
–Ј–∞–±—Л—В—Л–є `done()` = silent infinite wait.

Industry-standard: **closure-form** `call_once(fn)` вАФ single call,
exception/panic safe (RAII), –љ–µ–≤–Њ–Ј–Љ–Њ–ґ–љ–Њ misuse. Rust/Java/Kotlin вАФ
–≤—Б–µ –Є–Љ–µ—О—В closure form primary.

–Ф–Њ–њ–Њ–ї–љ–Є—В–µ–ї—М–љ—Л–µ gaps:

| Gap | Use case | Industry |
|---|---|---|
| **`call_once(closure)`** safe API | exactly-once init, panic-safe | Rust `Once::call_once`, Kotlin `synchronized`+flag, Java `AtomicInteger`-CAS |
| **`OnceCell[T]`** value-capturing init | lazy-init —Б capture value | Rust `OnceCell<T>` / `once_cell` crate / Rust 1.70+ stdlib |
| **`Lazy[T]`** auto-init on first access | global-style lazy constant | Rust `Lazy<T>` (once_cell), Kotlin `by lazy {}`, Java holder pattern |
| **`OnceGuard consume` (V2)** | explicit `.commit()` vs `.abort()` (e.g., conditional init) | Plan 100.7 future |

**M8 (master):** `call_once(closure)` primary; `run()/done()` deprecated
alias —Б migration warning.
**M9 (master):** OnceCell + Lazy вАФ distinct types (different APIs).

## Scope

### In scope

**`Once` (hardened):**
- **Add:** `call_once(body fn() -> ())` вАФ closure-form, primary API. –С–µ–Ј–Њ–њ–∞—Б–љ—Л–є
  –і–∞–ґ–µ –њ—А–Є panic –≤ body: cleanup —В—А–∞–љ–Ј–Є—В–Њ–Љ —З–µ—А–µ–Ј defer (mutex released,
  waiters wake —Б error state).
- **Add:** `is_completed() -> bool` (observability).
- **Deprecated:** existing `run() -> bool` / `done()` вАФ keep working,
  emit `W_ONCE_RUN_DONE_DEPRECATED` —З–µ—А–µ–Ј `compiler-codegen/src/diag.rs` (`Diagnostic` + `Suggestion` —Б `Applicability::MachineApplicable`), suggest
  `call_once(fn)`.
- **Panic-safety:** –µ—Б–ї–Є body panics inside `call_once`, Once transitions
  to `Poisoned` state; subsequent `call_once` calls also panic —Б
  `OncePoisoned` error. (Unlike `run()/done()` –≥–і–µ waiters –љ–∞–≤—Б–µ–≥–і–∞
  –Ј–∞–±–ї–Њ–Ї–Є—А–Њ–≤–∞–љ—Л.)

**`OnceCell[T]`** (value-capturing):
- `new() -> Self` (empty state)
- `get() -> Option[T]` (returns Some –µ—Б–ї–Є initialized; None –Є–љ–∞—З–µ)
- `set(v T) -> bool` (returns true –µ—Б–ї–Є this was the first set; false
  –µ—Б–ї–Є —Г–ґ–µ initialized вАФ value NOT replaced)
- `get_or_init(init fn() -> T) -> T` вАФ primary API. Lazy initialization
  using init closure, exactly-once across concurrent fibers. Subsequent
  calls return cached value –±–µ–Ј re-running init.
- `take() -> Option[T]` вАФ atomic remove value (returns Some –µ—Б–ї–Є –±—Л–ї
  set; resets cell back to empty). Useful –і–ї—П shutdown patterns.
- `is_initialized() -> bool` вАФ observability.

**`Lazy[T]`** (auto-init wrapper):
- `Lazy.new(init fn() -> T) -> Self` вАФ stores init closure, defers
  execution –і–Њ first access.
- `force() -> T` вАФ primary API. Triggers init if not yet, returns
  value. Idempotent (subsequent calls return cached).
- `is_forced() -> bool` вАФ observability.
- **Difference vs OnceCell:** Lazy hides init closure (one-shot setup
  pattern: ¬Ђ–≤–Њ—В —Д—Г–љ–Ї—Ж–Є—П, –Ї–Њ—В–Њ—А–∞—П —Б–Њ–Ј–і–∞—С—В –Ј–љ–∞—З–µ–љ–Є–µ, –і–∞–є –Љ–љ–µ –µ–≥–Њ¬ї);
  OnceCell allows set/get/get_or_init —Б different closures per call
  (more flexible).

### Out of scope

- **`OnceGuard consume`** (Plan 100.7-style commit/abort) вАФ V2 sub-plan
  103.9 (gated on Plan 100.7).
- **Global Lazy** (`static LAZY: Lazy<T>` Rust pattern) вАФ Nova
  doesn't have `static` globals; user puts `Lazy[T]` value –≤ module-
  level binding. Plan 103.5 design works –≤ –ї—О–±–Њ–Љ scope.
- **Concurrent-init coalescing** (multiple init closures concurrently вАФ
  only one runs, others wait) вАФ implicit –і–ї—П OnceCell.get_or_init (–њ–Њ
  semantics), –љ–µ —В—А–µ–±—Г–µ—В separate API.
- **`Lazy[T]` —Б custom strategy** (sync vs none vs publication вАФ Kotlin
  `LazyThreadSafetyMode`) вАФ V1 sync-only; future plan –µ—Б–ї–Є –љ—Г–ґ–љ–Њ.
- **OnceCell –±–µ–Ј panic-safety** (Rust `UnsyncOnceCell`) вАФ Nova
  –≤—Б–µ–≥–і–∞ sync (fiber-safe); single-threaded variant –љ–µ –љ—Г–ґ–µ–љ (sync
  primitives M:N-correct –њ–Њ –і–Є–Ј–∞–є–љ—Г).

## API design

### `Once` (hardened)

```nova
module runtime.sync

type OnceState {
    Fresh
  | Running
  | Done
  | Poisoned
}

/// Exactly-once execution barrier. Fiber-aware (parking).
///
/// # Primary API: `call_once(fn)` (M8)
///
/// Panic-safe: –µ—Б–ї–Є closure panics, Once transitions to `Poisoned`;
/// subsequent `call_once` also panic —Б `OncePoisoned` error.
///
/// # Legacy API: `run()/done()` вАФ deprecated since 0.2
///
/// Emit `W_ONCE_RUN_DONE_DEPRECATED`; suggest `call_once(fn)`.
/// –°–Њ—Е—А–∞–љ–µ–љ—Л –і–ї—П backward compat (M14).
#stable(since = "0.1")
export external fn Once.new() -> Self

/// PRIMARY API (M8). Closure-form exactly-once init.
///
/// # Panic-safety
/// If `body` panics, Once вЖТ `Poisoned`; subsequent calls re-panic.
///
/// # Examples
///
/// ```nova
/// let init = Once.new()
/// init.call_once { ||
///     expensive_setup()
/// }
/// // safe: subsequent calls no-op
/// init.call_once { ||
///     never_called_again()
/// }
/// ```
#stable(since = "0.1")
export external fn Once mut @call_once(body fn() -> ())

#stable(since = "0.1")
export external fn Once @is_completed() -> bool
#stable(since = "0.1")
export external fn Once @state() -> OnceState

// вФАвФАвФА Deprecated API вФАвФАвФА
#deprecated(since = "0.1", suggest = "call_once(fn)")
#stable(since = "0.1")
export external fn Once mut @run() -> bool

#deprecated(since = "0.1", suggest = "call_once(fn)")
#stable(since = "0.1")
export external fn Once mut @done()
```

### `OnceCell[T]`

```nova
/// Value-capturing one-shot initialization cell. Empty –њ—А–Є new();
/// transitions to initialized at first `set` / `get_or_init`.
///
/// # Use case
/// Cached computed value, lazy resource initialization (db connection,
/// config snapshot).
#stable(since = "0.1")
export external fn OnceCell[T].new() -> Self

#stable(since = "0.1")
export external fn OnceCell[T] @get() -> Option[T]

#stable(since = "0.1")
export external fn OnceCell[T] mut @set(v T) -> bool

/// PRIMARY API. Initializes cell if empty (using `init` closure),
/// returns value. Concurrent calls: only one init closure runs;
/// others wait for completion + receive cached value.
///
/// # Panic-safety
/// If `init` panics, cell remains empty; next call retries (unlike
/// Once.call_once which poisons).
#stable(since = "0.1")
export external fn OnceCell[T] mut @get_or_init(init fn() -> T) -> T

/// Atomically remove value. Returns Some if was initialized;
/// cell becomes empty (re-initializable).
#stable(since = "0.1")
export external fn OnceCell[T] mut @take() -> Option[T]

#stable(since = "0.1")
export external fn OnceCell[T] @is_initialized() -> bool
```

### `Lazy[T]`

```nova
/// Auto-init wrapper. Stores init closure, deferred execution until
/// first `force()`. Result cached, idempotent.
///
/// # Vs OnceCell
/// Lazy hides init: closure provided at construction (one-shot pattern).
/// OnceCell allows different closures per get_or_init call (flexible).
/// Use Lazy –і–ї—П ¬Ђ–≤–Њ—В –Ј–љ–∞—З–µ–љ–Є–µ, –ї–µ–љ–Є–≤–Њ –Є–љ–Є—Ж–Є–∞–ї–Є–Ј–Є—А—Г–µ—В—Б—П¬ї; OnceCell –і–ї—П
/// ¬Ђcache slot —Б different init strategies¬ї.
#stable(since = "0.1")
export external fn Lazy[T].new(init fn() -> T) -> Self

/// PRIMARY API. Triggers init if not yet done, returns cached value.
///
/// # Panic-safety
/// If init panics, Lazy вЖТ `Poisoned`; subsequent `force()` re-panic.
#stable(since = "0.1")
export external fn Lazy[T] mut @force() -> T

#stable(since = "0.1")
export external fn Lazy[T] @is_forced() -> bool
```

### –§.1 вАФ `Once.call_once(fn)` + Poisoned state (~0.3d)

- Extend `Nova_Once` struct: add `NOVA_ONCE_POISONED = 3` state.
- `call_once(body)` impl:
  - Transition NEW вЖТ RUNNING (CAS).
  - Execute body within try/catch:
    - success вЖТ DONE (release-store); wake all waiters.
    - panic вЖТ POISONED (release-store); wake all waiters —Б error
      throw on resume.
  - Pattern matches Rust `Once::call_once` panic-safety.
- `is_completed() / state()` getters.
- Backward compat: `run()/done()` legacy API —Б–Њ—Е—А–∞–љ–µ–љ—Л, emit
  `W_ONCE_RUN_DONE_DEPRECATED` (lint registered).

Acceptance: 4 positive (call_once exactly-once across fibers,
state observable, deprecation warning emitted, panic poisons).

### –§.2 вАФ `OnceCell[T]` impl (~0.4d)

- `nova_rt/sync_primitives.h`: `Nova_OnceCell` struct (Once internal +
  value slot + value-set flag).
- `get() -> Option[T]`: load value flag (acquire); if set вЖТ return
  Some(value); else None.
- `set(v) -> bool`: CAS-style first-writer-wins; if was-empty вЖТ store
  value (release), return true; if was-set вЖТ return false (value
  unchanged).
- `get_or_init(init)`: combine Once.call_once + set:
  - If is_initialized вЖТ return cached.
  - Else use internal Once: call_once { value = init(); set_flag = true }.
  - After Once вЖТ return value (must be set).
  - Panic in `init`: cell remains empty (NOT poisoned; can retry).
- `take() -> Option[T]`: atomic test-and-clear: if was-set вЖТ return
  Some(value), clear flag; else None.
- `Nova_OnceCell_T` typedef per-T (mono'd by Plan 99 routing pattern).

Acceptance: 4 positive (get_or_init exactly-once, set returns false on
second call, take resets, panic in init retries).

### –§.3 вАФ `Lazy[T]` impl (~0.3d)

- `nova_rt/sync_primitives.h`: `Nova_Lazy` struct (OnceCell internal +
  init closure ref).
- `Lazy.new(init)`: store closure, OnceCell uninit.
- `force()`: `OnceCell.get_or_init(stored_init_closure)`; panic-on-init
  poisons Lazy (NOT OnceCell), subsequent force re-panic.
- `is_forced() -> bool`: delegate to OnceCell.is_initialized.

**Note:** Lazy = thin wrapper over OnceCell + saved init closure. The
poison semantics differ (Lazy panics on subsequent force after init
panic; OnceCell allows retry —Б new init).

Acceptance: 3 positive (force lazy, idempotent, poison on panic).
