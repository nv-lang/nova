# Промпты для агентов

Этот каталог хранит **рабочие промпты** для повторяемых операций над
проектом nova-lang. Промпты — отдельные от планов: план описывает
**что и зачем**, промпт — **как именно прогнать конкретную операцию**.

## Когда использовать

Промпт пишется когда:
- Операция повторяется (регенерация, sweep, миграция).
- Логика операции стабильна, может выполняться много раз.
- Хочется чтобы любой агент / разработчик мог прогнать её одинаково.

## Что внутри промпта

- Цель в одном абзаце.
- Точные входы (какие файлы / реестры читать) и выходы (что писать).
- Шаги: numbered list, без двусмысленностей.
- Acceptance: как проверить что прогон успешен.
- Ограничения: что **не** делать (например, не править вручную).
